# MU5250 API Template

模板 ID：`MU5250`

匹配规则：

- `device.model_name == "MU5250"`
- 或 `device.hardware_version` 以 `MU5250_` 开头

这个模板代表当前后端正式支持的 `U60 Pro` 设备侧接口集合。它的目标是把 `U60` 自己的接口固定下来，不再把其他机型的回退逻辑默认混进主路径。

## Device-Side Interfaces

`ubus`：

- `zte_nwinfo_api nwinfo_get_netinfo`
- `zwrt_bsp.battery list`
- `zwrt_bsp.charger list`
- `zwrt_bsp.thermal get_cpu_temp`
- `zwrt_router.api router_get_user_list_num`
- `zwrt_router.api router_get_status_no_auth`
- `zwrt_data get_wwandst`
- `zwrt_zte_mdm.api get_zwrt_common_info`
- `zwrt_zte_mdm.api get_imei`
- `zwrt_bsp.usb list`
- `zwrt_nfc zwrt_nfc_wifi_get`
- `zwrt_wms zwrt_wms_get_wms_capacity`
- `zwrt_wms zte_libwms_get_sms_data`
- `system info`
- `system board`

`uci`：

- `wireless.main_2g.ssid`
- `wireless.main_2g.key`
- `wireless.main_2g.encryption`
- `wireless.main_2g.disabled`
- `network.lan.ipaddr`
- `dhcp.lan.start`
- `dhcp.lan.limit`
- `dhcp.lan.leasetime`

文件 / sysfs：

- `/tmp/dhcp.leases`
- `/data/logfs/key.log`
- `/proc/stat`
- `/sys/class/power_supply/usb/voltage_now`
- `/sys/class/power_supply/usb/current_now`
- `/sys/class/power_supply/battery/voltage_now`
- `/sys/class/power_supply/battery/current_now`

## Field Mapping

| snapshot | source |
|----------|--------|
| `net.*` | `zte_nwinfo_api nwinfo_get_netinfo` |
| `net.nrca` / `net.lteca` | `zte_nwinfo_api nwinfo_get_netinfo`（载波聚合描述符，原样透传） |
| `net.wan_status` | `zwrt_router.api router_get_status_no_auth` |
| `wlan.ssid/key/enc` | `uci get wireless.main_2g.{ssid,key,encryption}` |
| `wlan.enabled` | `uci get wireless.main_2g.disabled`（取反） |
| `battery.*` | `zwrt_bsp.battery list` |
| `battery.charging/charger_*` | `zwrt_bsp.charger list` |
| `battery.chg_uv/chg_ua` | `/sys/class/power_supply/usb/voltage_now`、`current_now` |
| `battery.bat_uv/bat_ua` | `/sys/class/power_supply/battery/voltage_now`、`current_now` |
| `clients.total/wifi/lan` | `zwrt_router.api router_get_user_list_num` |
| `clients.list` | `/tmp/dhcp.leases` |
| `sms.unread` | `zwrt_wms_get_wms_capacity` 的 `sms_dev_unread_num + sms_sim_unread_num` |
| `sms.list` | `zte_libwms_get_sms_data {page:0,data_per_page:32,mem_store:1,tags:10,order_by:"order by id desc"}` |
| `nfc.switch` | `zwrt_nfc zwrt_nfc_wifi_get` |
| `traffic.*` | `zwrt_data get_wwandst {"source_module":"deviceui","cid":1,"type":1}` |
| `qos.usb_mode` | `zwrt_bsp.usb list` |
| `qos.qci` / `qos.ambr_*` | 启动时全量扫描 `/data/logfs/key.log` 的 QoS `[DATA]` 日志并缓存 |
| `dhcp.*` | `uci`：`network.lan.ipaddr`、`dhcp.lan.start/limit/leasetime` |
| `system.uptime/mem_*` | `system info` |
| `system.cpu_usage` | `/proc/stat` |
| `system.cpu_temp` | `zwrt_bsp.thermal get_cpu_temp` 的 `cpuss_temp` |
| `system.model/hostname/fw` | `system board` |
| `system.sw_version` | `zwrt_zte_mdm.api get_zwrt_common_info` 的 `wa_inner_version`，缺失时回退 `integrate_version` |
| `system.imei` | `zwrt_zte_mdm.api get_imei` |
| `device.model_name/hardware_version/market_name/alias_name` | `zwrt_zte_mdm.api get_zwrt_common_info` |
| `device.board_name` | `system board` |
| `device.profile` | 基于 `device.model_name` 规范化生成 |
| `device.api_template` | 后端模板选择结果，当前固定为 `MU5250` |

## Explicit Non-Goals

这个模板当前故意不启用下面这些“宽松兼容”来源：

- `zwrt_bsp.thermal list`
- `thermal_zone` sysfs CPU 温度回退
- `wireless.main_5g` 自动选主 Wi-Fi
- `zwrt_router.api router_lan_access_list`
- `zwrt_router.api router_wireless_access_list`

这些路径后续如果要服务 `G5Pro` 或其他机型，会放进对应机型模板里单独维护，而不是继续混在 `U60` 模板主路径里。
